Дополнительная информация о файлах:
 
- *_meta.csv - метаданные про каждую запись, взятые из официального репозитория https://physionet.org/content/ptb-xl/1.0.3/ (к ним добавлены номера групп врачей, которые размечали данные)
- EDA.ipynb - explanatory data analisys и создание файлов annotations_*.csv для последующей тренировки.
- transforming.ipynb - код препроцессинга файлов экг(получение удара сердца).
- model_training.ipynb - код для тренировки модели
